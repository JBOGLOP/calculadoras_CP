1 · Configuración inicial

Esta herramienta analiza artículos científicos en PDF aplicando el marco de análisis crítico de 10 dimensiones de la Maestría. Se conecta a la API de Anthropic con tu propia clave (no se almacena en el servidor) y registra cada análisis en Google Sheets.

¿Cómo obtengo una API Key de Anthropic?
  1. Ingresa a console.anthropic.com y crea cuenta (separada de tu suscripción Pro de Claude.ai).
  2. Añade un método de pago. La API se factura por uso, no por suscripción.
  3. En Settings → Limits, configura un monthly spend limit de USD 20–50 con alerta a tu correo.
  4. En Settings → API Keys, crea una clave nueva con nombre analizador-articulos-uan. Cópiala y pégala arriba.
  5. Guarda la clave en un gestor de contraseñas. Si la pierdes, deberás generar una nueva (no se puede recuperar).

Costo estimado por análisis: entre USD 0,30 y 1,50 según modelo y tamaño del PDF.

Sonnet 4.5 es la mejor opción para análisis crítico. Opus 4.7 para análisis publicables. Evita Haiku para análisis estructurados: en pruebas omitió bloques JSON en varias dimensiones.
Pega aquí la URL /exec de tu Web App de Apps Script si quieres guardar análisis en Google Sheets. Déjalo vacío para solo descarga local.

2 · Cargar artículo en PDF

Arrastra un artículo en formato PDF, o haz clic para seleccionar uno desde tu equipo. El PDF se envía directamente a la API de Anthropic (que lo lee nativamente, incluyendo tablas y figuras). Tamaño máximo: 32 MB / aprox. 100 páginas.
